01. Sobre mim

Quem sou eu

Desenvolvedor Full-Stack com paixão por resolver problemas complexos através de código limpo, escalável e acessível. Tenho experiência construindo aplicações web do zero — desde a arquitetura de banco de dados até a interface final que o usuário vê.

Acredito que o melhor software é aquele que o usuário nem percebe que está usando, porque simplesmente funciona. Por isso, une performance técnica com design intencional em cada projeto.

Fora do código, estou aprendendo sobre sistemas distribuídos, contribuindo para projetos open source e explorando novas tecnologias.
